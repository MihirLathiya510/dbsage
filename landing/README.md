# dbsage landing page

SvelteKit 5 + Tailwind v4 static landing page for dbsage.

## Setup

```sh
pnpm install
```

## Developing

```sh
pnpm dev

# open in browser automatically
pnpm dev -- --open
```

## Type checking

```sh
pnpm check
```

## Testing

```sh
pnpm test
```

## Building

```sh
pnpm build
```

Preview the production build:

```sh
pnpm preview
```

The build output goes to `build/` (static adapter — no server required).
